# AI Interview Simulator

A premium AI-powered interview simulation platform built with Next.js, Tailwind CSS, and Framer Motion. Uses local NLP algorithms (Cosine Similarity & TF-IDF) to analyze responses without external paid APIs.

## Features
- 🤖 **AI Analysis**: Real-time feedback using NLP keyword matching and cosine similarity.
- 🎨 **Premium UI**: Glassmorphism design with smooth Framer Motion animations.
- 🌓 **Dark Mode**: Fully responsive dashboard with theme switching.
- 🚀 **Vercel Ready**: Optimized for the Vercel free plan.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript

## Getting Started

1. **Clone the repo**
   ```bash
   git clone <your-repo-url>
   cd ai-interview-simulator
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Deploy to Vercel**
   - Push to GitHub.
   - Import to Vercel.
   - It works out of the box!
